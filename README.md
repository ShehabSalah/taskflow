# TaskFlow

**Context-aware, local-first work management for professionals.**

TaskFlow is a desktop application for tracking time, earnings, projects, and tasks across clients, employers, and contracts. Workspaces keep each context separate; your data is stored locally on your machine.

**[Website](https://shehabsalah.github.io/taskflow/)** · **[User guide](https://shehabsalah.github.io/taskflow/docs.html)** · **[Releases](https://shehabsalah.github.io/taskflow/#releases)**

## About this repository

This repository hosts the **public website** for TaskFlow: product overview, download information, release history, FAQ, and web documentation. It is a static site (HTML, Tailwind CDN, no build step).

The application itself lives in separate repositories (see below).

## What TaskFlow does

- **Workspaces** — one context per client, employer, or contract (salary, hours, overtime settings).
- **Projects & tasks** — Kanban boards, labels, checklists, attachments, and rich task details.
- **Time tracking** — timers at workspace, project, or task level; work logs with filters.
- **Earnings & overtime** — real-time pay from recorded sessions; daily, weekly, and monthly overtime views.
- **Reports** — Excel and CSV exports from actual time logs (timesheets, attendance, overtime).
- **Resources** — local files and folders tied to workspaces and projects.
- **Desktop** — native installers for Windows and macOS, with bundled or bring-your-own PostgreSQL.

TaskFlow is designed to run **offline**. There is no required cloud account for core features.
